Objective

The objective of this lab and assignment is to analyze stock returns using histograms to interpret distribution, then calculate and evaluate alpha, beta, and correlation for more complex stock selection.

Lab Task: Histogram Analysis (In-Class)

In this lab section, you will focus on analyzing stock distributions using histograms and making decisions based on skewness and kurtosis.

  • Task 0: Load Stock Data with get_stock_data (get_local_data.py)
    • Use the provided helper function download_local(symbols, start, end, data_dir="data") to read Adjusted Close prices from the CSV files in your data/ folder (no Internet required).
    • Date range for the warm-up: '2020-01-01' to '2022-12-31'.
    • Pick stocks not shown in lecture, save the resulting DataFrame to CSV, and print the first 10 rows to verify.
  • Task 1: Analyze Stock Distributions Using Histograms
    • Create histograms of the daily returns for each stock and compare them to SPY.
    • Overlay a Gaussian curve on each histogram to assess normality.
    • Calculate skewness and kurtosis for every stock; include the values in the legend.
  • Task 2: Pick Stocks Based on Histogram Analysis
    • Pick 2 stocks that exhibit favorable behavior based on their histogram analysis (e.g., closer to normal distribution with moderate skewness/kurtosis).
    • Pick 1 stock that has poor performance based on the histogram analysis (e.g., highly skewed or kurtotic distribution).

Homework: Correlation, Alpha, Beta, & Comparison Analysis

In the homework portion, you will use correlation, alpha, and beta to further refine your stock choices from the lab section. You will compare the picks made based on the histograms against those selected using correlation and alpha/beta analysis.

You will do the following tasks twice, once with the stocks you picked in the lab, and once by considering a larger set of stocks, see below in & " Follow-Up Analysis."

  • Task 3: Scatter Plot and Correlation Analysis (scatterplot_analysis.py)
    • For each of the 3 stocks you picked in the lab section, plot a scatter plot showing the daily returns compared to SPY.
    • Fit a linear regression line and calculate the correlation coefficient.
    • Use this correlation analysis to assess the relationship of each stock to the market.
  • Task 4: Alpha & Beta Analysis
    • Calculate the alpha and beta for each of your selected stocks using SPY as the market benchmark.
    • Interpret the alpha (excess returns) and beta (volatility relative to the market) to decide which stock has performed better or worse in comparison to SPY.
  • Task 5: Compare Stocks Chosen by Histogram vs. Alpha/Beta Analysis
    • Compare the stocks you selected in the lab based on histograms with the stocks you would select based on alpha/beta and correlation.
    • Discuss which method (histogram analysis vs. alpha/beta analysis) you found to be more effective or insightful.

Follow-Up Analysis: Picking Stocks for a Full Portfolio

In this follow-up assignment, you'll extend the analysis to a larger set of stocks.

  1. Load the historical prices for all 500 S&P 500 companies from the CSV already included in the data/ folder (downloaded in 2024 – no Internet/yfinance needed).
  2. Download historical prices for all 500 S&P 500 companies for the selected year or time period (CSV provided - see schedule page link).
  3. Focus only on adjusted close prices to account for dividends and stock splits.
  4. For each stock, compute the daily return for each day.
  5. Repeat this calculation for your benchmark (most likely SPY).
  6. Select two stocks, for example, JPM (JPMorgan Chase) and NVDA (NVIDIA).
  7. For each of these two stocks, create a scatter plot of SPY (benchmark) daily returns versus the stock’s daily returns. Fit a line to the data points and display it in red.
  8. For each of the two stocks, calculate the following:
    • Beta
    • Alpha
    • Pearson’s correlation coefficient
  9. Create a chart that includes histograms of the daily returns for all three: SPY, JPM, and NVDA. Use reduced opacity to overlay the histograms. Draw a vertical line for each asset at its mean daily return.
  10. Consider the plot and describe how Beta, Alpha, and relative volatility are represented visually.

Finally, compute the Beta, Alpha, and standard deviation (volatility) for all 500 S&P 500 stocks. Answer the following questions:

  • Is Beta correlated with the standard deviation of returns (volatility)?
  • Is Alpha correlated with the standard deviation?
